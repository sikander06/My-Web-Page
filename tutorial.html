How to change your IP address and DNS server

In this tutorial, I will show you how to change your IP address and DNS server in only 8 steps


Step 1: Windows Start Menu

To access the Windows Start Menu, click on the Windows logo located at the left-hand side of your taskbar.


Step 2: Settings

Once you have accessed the Windows Start Menu, you may locate the Settings app either by performing a search or by locating its icon.
Another convenient way to open Settings is to use the keyboard shortcut ‘Windows logo key + I’.


Step 3: Network Status

To proceed with changing your IP address and DNS server, the next step is to access the Settings app and type "Network Status" in the search bar to locate the corresponding section.


Step 4: Status & Tasks

In the subsequent step, we need to access the "Change adapter options" section. To locate it, we must first click on "View network status and tasks".


Step 5: Change Adapter Options

Once you have accessed the "Change adapter options" section, please locate the network with the green "staircase" in the right corner and right-click on it.


Step 6: Properties

After right-clicking on the network you wish to modify, the subsequent step is to click on "Properties" located at the bottom of the menu.


Step 7: Internet Protocol Version 4

Once you have clicked on "Properties", a new menu will appear. Next, you should scroll down and select "Internet Protocol Version 4" and click on its properties.

Note that if clicking on "Properties" and then "OK" does not work, you can alternatively double-click on "Internet Protocol Version 4".


Step 8: IP Address & DNS Server

At this point, you should be able to see the section where you can modify the IP and DNS server addresses. To proceed, click on "Use the following IP address".


After filling in all the required fields, click on "OK" to save the changes. Congratulations, you have successfully configured a new IP address and DNS server!
